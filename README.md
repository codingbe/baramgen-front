## ✨ Feature
- 시간에 따른 자동 계산기능이 있습니다.
- 종료후 다시 시작해도 기록이 남아있습니다
- 기록을 한 뒤 홈에서 다음 젠 시간이 제일 빠른순으로 볼 수 있습니다
- 최근에 잡은 시간 또는 최근 점검 시간 기준으로 예상 젠 시간을 계산해 줍니다
- 반응형 웹을 구현하여 모바일과 PC에서 편리하게 이용할 수 있습니다
- 커뮤니티를 통해 소통할 수 있습니다.
